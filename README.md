# Notulensi: 100% Offline Private Meeting Assistant

Notulensi is a privacy-first mobile application built with Flutter that transcribes and structures meeting notes entirely on-device. No data ever leaves your phone.

## 🚀 Vision 2027
Capturing the future of private professional collaboration through hardware-integrated, biometric-secured, and intelligently structured offline notes.

## 🛠 Tech Stack
- **Frontend**: Flutter (BLoc/Cubit)
- **Database**: Isar (Encrypted, Full-Text Search)
- **Intelligence**: Local OS STT + Vosk Pro, Isolate-based Regex Extraction
- **Security**: AES-GCM 256, Biometric Safe-Box (FaceID/Fingerprint)
- **Monetization**: AdMob Rewards (Pro Templates)
