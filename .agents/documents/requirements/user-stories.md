# User Stories: Notulensi

## Core: Recording & Transcription
- **US-001 (Record Audio):** As a professional in a meeting, I want to record audio directly within the app so that I can capture the full conversation without needing external recording devices.
- **US-002 (Real-time Transcription):** As a user, I want to see a live transcript of the audio being recorded so that I can monitor the capture accuracy in real-time.
- **US-003 (Stop & Save):** As a user, I want to stop the recording and have the audio and transcript saved immediately so that I don't lose any data.
- **US-004 (Offline Operation):** As a privacy-conscious user, I want all recording and transcription to happen 100% offline so that my sensitive data never leaves my device.

## Note Management & Organization
- **US-005 (List Notes):** As a regular user, I want to see a list of all my saved meeting notes so that I can easily find and review previous sessions.
- **US-006 (Note Details):** As a user, I want to open a specific note to view its full transcript and structured highlights so that I can retrieve the information I need.
- **US-007 (Search Notes):** As a user with many meetings, I want to search through my notes by title or content so that I can quickly locate specific information.
- **US-008 (Edit Note):** As a user, I want to edit the title or transcript of a note to correct any transcription errors or add context.
- **US-009 (Delete Note):** As a user, I want to delete notes that are no longer needed so that I can keep my local storage organized.
- **US-025 (Project Folders):** As a consultant working with multiple clients, I want to organize notes into folders so that I can keep different projects separated.
- **US-026 (Note Versioning):** As a user who frequently refines transcripts, I want to see a history of my manual edits so that I can revert to previous versions if needed.

## Security & Privacy
- **US-017 (Physical Safe-Box):** As a high-security user, I want to lock the app with my fingerprint or face ID so that my notes are protected even if someone else holds my phone.
- **US-020 (Privacy Masking):** As a legal professional, I want the app to automatically censor names and numbers in a transcript before I export it so that I can maintain confidentiality.
- **US-028 (Encrypted Local Backup):** As a user concerned about data loss, I want to back up my encrypted database to my SD card so that I have a local copy outside the app sandbox.

## Smart Interaction & Multitasking
- **US-018 (Waveform Markers):** As a user listening to a long meeting, I want to drop "stars" on the audio waveform during recording so that I can quickly find important moments later.
- **US-024 (Voice Commands):** As a user with my hands full, I want to say "Stop Recording" to end a session so that I don't have to touch the screen.
- **US-029 (Physical Key Markers):** As a user recording with my phone in my pocket, I want to press the volume buttons to mark important points so that I don't draw attention to my phone.
- **US-030 (Smartwatch Remote):** As a smartwatch user, I want to control the recording from my wrist so that I can keep my phone hidden.
- **US-035 (PiP Transcript):** As a multitasker, I want to see the live transcript in a small window while using other apps so that I can take notes or do research simultaneously.

## Advanced Intelligence (Local Only)
- **US-010 (Action Item Extraction):** As a project manager, I want the app to automatically highlight action items based on keywords so that I can track follow-up tasks without manual review.
- **US-011 (Deadline Extraction):** As a busy professional, I want the app to identify and extract deadlines mentioned in the meeting so that I can stay on top of my schedule.
- **US-019 (Speaker Tagging):** As a user in a group discussion, I want the app to detect when a different person starts speaking so that the transcript is broken into readable paragraphs.
- **US-021 (Calendar Sync):** As a user who tracks deadlines, I want to add a detected deadline to my phone's calendar with one tap so that I don't have to type it manually.
- **US-022 (Context Linking):** As a power user, I want the app to show me links to previous notes that mention the same keywords so that I can build a connected knowledge base.
- **US-023 (Noise Suppression):** As a user in a noisy cafe, I want the app to filter out background noise so that the transcription accuracy remains high.

## Multimedia & Analytics
- **US-031 (Meeting Photos):** As a student, I want to take a photo of the whiteboard during a recording so that the image is anchored to the exact time in the transcript.
- **US-032 (Silence Trimmer):** As a user concerned about storage, I want the app to automatically remove long silences from the audio file so that I can save space.
- **US-033 (Speech Analytics):** As a public speaker, I want to see a chart of my speaking speed (WPM) so that I can improve my presentation delivery.

## Monetization & Portability
- **US-013 (Export TXT):** As a user, I want to export my notes as a plain text file so that I can share them via offline methods.
- **US-014 (Export PDF):** As a consultant, I want to export a professional PDF summary of the meeting so that I can provide it to my clients.
- **US-034 (Reward Unlocks):** As a free user, I am willing to watch a reward ad to unlock premium PDF templates or privacy features for a day.
- **US-016 (Ad-Supported Usage):** As a free user, I am willing to see unobtrusive banner ads so that I can use the app for free.
