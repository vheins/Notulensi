# User Stories: Notulensi

## Core: Recording & Transcription
- **US-001 (Record Audio):** As a professional in a meeting, I want to record audio directly within the app so that I can capture the full conversation without needing external recording devices.
- **US-002 (Real-time Transcription):** As a user, I want to see a live transcript of the audio being recorded so that I can monitor the capture accuracy in real-time.
- **US-003 (Stop & Save):** As a user, I want to stop the recording and have the audio and transcript saved immediately so that I don't lose any data.
- **US-004 (Offline Operation):** As a privacy-conscious user, I want all recording and transcription to happen 100% offline so that my sensitive data never leaves my device.

## Note Management
- **US-005 (List Notes):** As a regular user, I want to see a list of all my saved meeting notes so that I can easily find and review previous sessions.
- **US-006 (Note Details):** As a user, I want to open a specific note to view its full transcript and structured highlights so that I can retrieve the information I need.
- **US-007 (Search Notes):** As a user with many meetings, I want to search through my notes by title or content so that I can quickly locate specific information.
- **US-008 (Edit Note):** As a user, I want to edit the title or transcript of a note to correct any transcription errors or add context.
- **US-009 (Delete Note):** As a user, I want to delete notes that are no longer needed so that I can keep my local storage organized.

## Structure & Extraction
- **US-010 (Action Item Extraction):** As a project manager, I want the app to automatically highlight action items based on keywords so that I can track follow-up tasks without manual review.
- **US-011 (Deadline Extraction):** As a busy professional, I want the app to identify and extract deadlines mentioned in the meeting so that I can stay on top of my schedule.
- **US-012 (Manual Tagging):** As a user, I want to manually tag specific parts of the transcript as important so that I can customize the summary.

## Export & Portability
- **US-013 (Export TXT):** As a user, I want to export my notes as a plain text file so that I can share them via offline methods or use them in other applications.
- **US-014 (Export PDF):** As a consultant, I want to export a professional PDF summary of the meeting so that I can provide it to my clients.

## Settings & Privacy
- **US-015 (Manage Storage):** As a user with limited phone storage, I want to see how much space my recordings are taking so that I can manage my device effectively.
- **US-016 (Ad-Supported Usage):** As a free user, I am willing to see unobtrusive banner ads so that I can use the app for free while supporting the developer.
