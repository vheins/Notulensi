# Memory Entry Template

Copy this template when adding a new entry to `.agents/memory.md`.

```markdown
## [YYYY-MM-DD] <Type>: <Title>

**Type:** decision_made | new_pattern | config_change | rule_established
**Context:** Why this entry exists — what problem or situation prompted it
**Decision/Pattern:** What was decided or established (be specific)
**Impact:**
- What files/modules are affected
- What behavior changes
**Do NOT:** The most common mistake to avoid related to this entry
```

---

## Change Types Reference

| Type | Use When |
|------|----------|
| `decision_made` | An architectural or technology decision was made |
| `new_pattern` | A new coding pattern was established for the project |
| `config_change` | A tool, environment, or configuration was changed |
| `rule_established` | A project-specific rule was agreed upon |
